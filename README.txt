To run console app from visual studio
Method 1
 1. Right click on solution file and click Multiple startup project
 2. Select action as start for both HelloWorldConsoleApp and HelloWorldAPI projects

Method 2
 1. Right click on  HelloWorldAPI project and set it as the startup project
 2.Then go to HelloWorld\HelloWorldConsoleApp\bin\Debug and double click HelloWorldConsoleApp.exe to run

Note: Ninject has used as a dependency injection framework,
 If it is not coming you can restore by going to Nuget package Manager which is under Visual studio Tools
